# Recipes

A website featuring recipes for various dishes.
Each recipe includes ingredients, instructions, and tips for preparation.
Feel free to explore and try them out in your kitchen!

Final assigment for the HTML section of [The Odin Project](https://www.theodinproject.com/).

Spec and details for the assignment can be found [here](https://www.theodinproject.com/lessons/foundations-recipes).

u can find the live version of the project [here](https://tasosbeast.github.io/odin-recipes/).
